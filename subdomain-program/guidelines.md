# Python Bangladesh Sub-Domain Program

[Python Bangladesh Organization](https://pythonbd.org/) is introducing a **Sub-Domain Program** to support and connect **like-minded tech communities** across Bangladesh, including (but not limited to) Django, AI/ML, Data Science, and university-based groups.

## üéØ Program Objective

The goal of this initiative is to build a **collaborative ecosystem** where independent communities can come together under the Python Bangladesh umbrella to:

- Foster collaboration and knowledge sharing  
- Co-organize workshops, meetups, events, and conferences  
- Strengthen the overall Python and open-source ecosystem in Bangladesh  

## üå± Community Values

Python Bangladesh follows the philosophy of the **[Python Software Foundation (PSF)](https://www.python.org/psf-landing/)** and promotes:

- Open Source  
- Open Access  
- Community-driven growth  

To align with these principles, participating communities must be **non-commercial and/or non-profit** in nature.

## üåê Sub-Domain Support

Approved communities will receive a dedicated sub-domain under `pythonbd.org`, for example:

- `ruet.pythonbd.org`  
- `aimlcommunity.pythonbd.org`  

These sub-domains are intended to provide visibility, credibility, and a shared identity within the national Python community.

---

## üìù How to Apply

Communities interested in joining the program should email **admin@pythonbd.org** with the following information:

- Organization or community name and brief description  
- Email addresses of **two primary contact persons**  
- Intended use of the sub-domain  
- Preferred sub-domain name  
- Brief technical requirements (if any)

## ‚úÖ Eligibility Requirements

To be considered, your community must:

- Have an **active website**  
  - A custom domain is not required  
  - GitHub Pages or free hosting platforms are acceptable  
- Clearly include the following sections on the website:
  - Activities  
  - Team  
  - Code of Conduct  
- Maintain a **Code of Conduct** aligned with the **[Python Bangladesh Code of Conduct](https://github.com/pythonbangladesh/public-docs/blob/main/code-of-conduct.md)**  

## üîç Review Process

All applications will be reviewed by the Python Bangladesh team. Approved communities will receive guidance on the most suitable technical and administrative setup based on their needs.
